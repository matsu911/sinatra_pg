Simple Sinatra Project Generatgor
=================================

Usage
-----
$ ./sinatra_pg.rb [project name]

Example
-------
$ ./sinatra_pg.rb sample_project  
$ tree -F sample_project  
sample_project  
|-- Gemfile  
|-- Rakefile  
|-- app.rb  
|-- config.ru  
|-- public/  
|-- test/  
`-- views/  
  
3 directories, 4 files
